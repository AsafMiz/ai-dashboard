Add a new example report to the dashboard: $ARGUMENTS

Steps:
1. Create a new JSON file at `public/data/example/reports/{report-key}.json` following the structure of existing reports (dashboard_key: "example", report_key, title, description, type, widgets array with inline data)
2. Each widget should have: type, title, config (without datasetKey — auto-generated by API), and data (array of row objects)
3. Widget order is determined by array position — no explicit order field needed
4. Add the report key to the `reports` array in `public/data/example/dashboard.json`
5. Update the reports table in `docs/guide.md` with the new report
6. Verify with `npx next build`
